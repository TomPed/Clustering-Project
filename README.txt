Running the program is simple.

  1. Open terminal
	2. type java -jar (insert jar here) (insert file here) k

	The jar is given to you in the /out/artifacts/ClassProject_jar
	folder, the file is in the root folder, and k is the number of
	clusters

	OR

	Open the your IDE of choice and enter, for the program args,
	the file and then the number of clusters and run the main
	method.
